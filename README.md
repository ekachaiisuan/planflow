## Project Tracking
-----
# Goal of this project
- เพื่อสร้างระบบติดตามความคืบหน้าของโครงการใช้งานภายในองค์กร 
- ทำระบบ workspace เพื่อใช้เป็นทีมแต่ละโครงการ
- ต้องสร้างการ์ดสําหรับแต่ละโครงการได้
- ต้องสามารถสร้าง report แยกโครงการได้ output เป็น pdf (แยกตามโครงการและรวมทุกโครงการ)
- เอาการ์ดแต่ละโครงการ มารวมเป็นแผนปฏิบัติการประจำปีงบประมาณ output เป็น pdf (ออกแบบตาม form ที่สำนักงานกำหนดไว้)
- เมื่อสร้างการ์ดโครงการแล้ว ต้องใช้ข้อมูลร่วมกันภายในทีม และทำระบบ sub task เพื่อบอกความก้าวหน้าโครงการได้
- ระบบมีความสามารถ upload file ได้
- ออกแบบ Dashboard ให้มีความสามารถติดตามความคืบหน้าของโครงการได้
- ออกแบบ Notification ให้มีความสามารถแจ้งเตือนโครงการใกล้ครบกำหนดได้ (แจ้งทาง web app และแจ้งทาง email)
- เมื่อจบปีงบประมาณต้องสามารถออกรายงานผลการดำเนินงานตามแผนปฏิบัติการได้ output เป็น pdf (ออกแบบตาม form ที่สำนักงานกำหนดไว้)
